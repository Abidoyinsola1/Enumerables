# Enumerables

This project uses our methods for Enumerables in Ruby.The methods created are:

- #my_each
- #my_each_with_index
- #my_select
- #my_all?
- #my_any?
- #my_none?
- #my_count
- #my_map
- #my_inject

## Built With

- RUBY

## Getting Started

**To view this project, download the contents of the repository to your computer or clone the repository and open the bubble_sort.rb file**
